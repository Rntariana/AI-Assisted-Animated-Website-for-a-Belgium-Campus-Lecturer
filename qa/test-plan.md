# Test Plan

## Project Overview
This project is an AI-assisted animated portfolio website developed for a Belgium Campus lecturer. The purpose of testing is to ensure the website functions correctly, is responsive across devices, and provides a smooth user experience.

## Testing Objectives
- Verify that all website features work as expected
- Ensure responsiveness across mobile, tablet, and desktop devices
- Identify and document bugs or UI issues
- Ensure animations perform smoothly

## Scope of Testing
### Included:
- Navigation and links
- Animations and transitions
- Layout responsiveness
- Cross-browser compatibility

### Excluded:
- Backend/database testing (if not applicable)

## Testing Types
- Manual Functional Testing
- UI/UX Testing
- Responsive Design Testing
- Cross-Browser Testing

## Tools Used
- Manual testing via browsers (Chrome, Edge)
- GitHub Issues for bug tracking

## Test Environment
- Devices: Laptop, Mobile phone
- Browsers: Google Chrome, Microsoft Edge
- OS: Windows

## Entry Criteria
- Website is fully developed and deployed

## Exit Criteria
- All major bugs are resolved
- Website performs correctly across tested devices and browsers
